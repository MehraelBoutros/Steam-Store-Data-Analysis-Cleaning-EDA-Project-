# 🎮 Steam Store Data Analysis & Cleaning

## 📌 Project Overview
This project focuses on performing full data cleaning and exploratory data analysis (EDA) on a Steam Store dataset. The goal is to understand patterns in game performance, player engagement, and user feedback.

---

## 🎯 Objectives
- Clean and preprocess raw dataset
- Handle missing values and outliers
- Analyze player behavior (playtime, ratings)
- Identify top-performing games
- Discover trends in positive/negative reviews

---

## 🧹 Data Cleaning Steps
- Removed duplicates
- Handled missing values
- Converted data types
- Detected and treated outliers using:
  - Boxplots
  - IQR method

---

## 📊 Exploratory Data Analysis (EDA)
- Distribution analysis of:
  - Positive ratings
  - Negative ratings
  - Average playtime
- Correlation analysis between variables
- Identifying:
  - Most popular games
  - Highly rated vs low-rated games
- Outlier detection and interpretation

---

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📈 Key Insights
- Games with higher playtime tend to have more positive engagement
- Some games have extreme outliers in playtime
- Strong relationship between positive and total ratings
